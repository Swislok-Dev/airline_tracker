# Airline Tracker

Track flights using a flight number

## Technology Use

_SvelteKit_
_FlightAware API_

## Uses

- Check current flight status
- Ability to track current progress of flight
- Arrows to search through past and future flight
- Show airport codes and city locations

## Future Plans

- Include weather options for airports
- Search for flight to and from given destinations
